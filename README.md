
# Divyansh Singh's Portfolio

[![Ask DeepWiki](https://devin.ai/assets/askdeepwiki.png)](https://deepwiki.com/pfldivyanshsingh/portfolio)

This repository contains the source code for my personal portfolio website. It is a fully responsive static website designed to showcase my skills, projects, educational background, and achievements. The site serves as a central hub for my professional online presence.

## ✨ Overview

This portfolio is a multi-page web application built with HTML, CSS, and JavaScript. It is designed to be clean, modern, and easy to navigate for visitors. The website is responsive and optimized for both desktop and mobile viewing, providing a seamless user experience across all devices.

The key sections include:
- A comprehensive **Home** page with a personal introduction.
- A detailed **Skills** page outlining my technical and professional abilities.
- A **Projects** section with descriptions and links to my work.
- An **Achievements** page listing certifications and extracurricular activities.
- A dedicated **Contact** page with multiple ways to get in touch.

## 🚀 Key Features

*   **Responsive Design**: The layout adapts to various screen sizes, from mobile phones to desktop monitors, using CSS media queries and flexible design principles.
*   **Multi-Page Layout**: The site is organized into logical sections, including Home, Skills, Projects, and Contact, for clear navigation.
*   **Interactive UI**: Subtle animations and hover effects, implemented with CSS transitions and JavaScript's `IntersectionObserver` API, enhance the user experience.
*   **Introduction and Bio**: A welcoming introduction on the main page details my background as a Computer Science and Engineering student with a passion for web development.
*   **Skills Showcase**: Displays technical skills such as HTML, CSS, JavaScript, Python, and C++, along with professional skills like team management and problem-solving.
*   **Project Gallery**: Features key projects, including this portfolio itself, with links to live versions and GitHub repositories.
*   **Contact & Social Links**: Provides clear contact information, including email and links to my LinkedIn, GitHub, and Instagram profiles.

## 🛠️ Tech Stack

This portfolio is built using foundational web technologies:

*   **HTML5**: For the structure and semantic markup of the web pages.
*   **CSS3**: For styling, layout (using Flexbox and Grid), and responsiveness.
*   **JavaScript**: For dynamic fade-in animations on scroll and interactive elements.

## 🌐 Local Setup and Usage

To run this portfolio website on your local machine, follow these simple steps:

1.  **Clone the repository:**
    ```sh
    git clone https://github.com/pfldivyanshsingh/portfolio.git
    ```

2.  **Navigate to the project directory:**
    ```sh
    cd portfolio
    ```

3.  **Open the `index.html` file in your web browser:**
    You can do this by double-clicking the file in your file explorer or right-clicking and selecting "Open with" your preferred browser.
